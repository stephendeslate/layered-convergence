## 13. Component Library Notes

All UI components use **shadcn/ui** with **Tailwind CSS 4**:

| Component | shadcn Component | Usage |
|-----------|-----------------|-------|
| Demo Banner | Alert | Warning variant, full-width |
| Navigation | NavigationMenu | Top nav with role-based items |
| Data Tables | Table | Transaction lists, dispute queue |
| Cards | Card | Metric cards, detail panels |
| Forms | Form + Input | Payment creation, dispute filing |
| Timeline | Custom | State history visualization |
| Charts | Recharts | Volume, fees, status distribution |
| Buttons | Button | Primary, secondary, destructive |
| Badges | Badge | Status indicators |
| Dialogs | Dialog | Confirmation modals |
| Select | Select | Dropdown menus |
| Toast | Toast | Success/error notifications |
